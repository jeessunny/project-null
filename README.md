## 🧩 Overview

**PROJECT NULL** is a browser-based puzzle experience designed to test:

- 🔍 Logical thinking
- 🔄 Pattern recognition
- 💡 Basic programming knowledge

| # | Title | Concept |

---

## ⚙️ How It Works

### 🔢 Level 1 — Number Logic
Identify a pattern and find the missing value.

### 💾 Level 2 — Broken Binary
Fix incomplete binary values and decode them into text.

### 🖥️ Level 3 — C Puzzle
Analyze a C program and determine its output using indexing and ASCII logic.

### 🔑 Level 4 — Cipher Shift
Decode an encrypted word using a Caesar cipher (shift backward).
